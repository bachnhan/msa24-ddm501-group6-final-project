# Team Contributions & Roles

This project is a collaborative effort for the **DDM501 - AI in Production** Final Project.

## 👥 Team Members

### 1. Lê Huỳnh Trang

- **Role**: Business Analyst / Project Manager
- **Responsibilities**:
  - Problem definition, business context formulation, and scope definition.
  - System architecture design and technical documentation (`ARCHITECTURE.md`).
  - Data Flow Diagram and System Design documentation.
  - Success metrics definition across Business, Model, and System levels.
  - Project documentation: `README.md`, `CONTRIBUTING.md`, and `docs/` structure.

### 2. Đỗ Trọng Minh Quân

- **Role**: ML Engineer / Researcher
- **Responsibilities**:
  - Model training, evaluation, and hyperparameter tuning.
  - Experiment tracking implementation (MLflow Integration).
  - Data quality testing and model validation.
  - Project documentation and user manuals.

### 3. Nguyễn Huỳnh Bách Nhân

- **Role**: DevOps / MLOps Engineer
- **Responsibilities**:
  - System architecture design and technical stack justification.
  - FastAPI development (REST API Design).
  - Containerization and Orchestration (Docker/Docker Compose).
  - Monitoring stack (Prometheus/Grafana) and CI/CD Pipeline.

## 🤝 Collaboration Workflow

- **Branching Strategy**: GitFlow (develop for features, main for production).
- **Communication**: Discord / In-person sessions.
- **Code Review**: Each PR requires at least one approval from a different member.
